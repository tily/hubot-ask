# hubot-ask

A hubot script to ask questions

See [`src/ask.coffee`](src/ask.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-ask --save`

Then add **hubot-ask** to your `external-scripts.json`:

```json
["hubot-ask"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
